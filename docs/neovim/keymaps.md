# Keymaps

_Muscle memory_

## Leader Keys

| Key     | Role         |
| ------- | ------------ |
| `Space` | Leader       |
| `,`     | Local leader |

## File Navigation

| Key         | Action               |
| ----------- | -------------------- |
| `Space f f` | Find files           |
| `Space f w` | Find word (grep)     |
| `Space f r` | Recent files         |
| `Space f b` | Find buffers         |
| `Space e`   | Toggle file explorer |

## Buffers

| Key         | Action              |
| ----------- | ------------------- |
| `]b`        | Next buffer         |
| `[b`        | Previous buffer     |
| `Space b d` | Delete buffer       |
| `Space b D` | Force delete buffer |

## Windows

| Key            | Action           |
| -------------- | ---------------- |
| `Ctrl h/j/k/l` | Navigate windows |
| `Space w`      | Window menu      |

## LSP

| Key         | Action                |
| ----------- | --------------------- |
| `gd`        | Go to definition      |
| `gD`        | Go to declaration     |
| `gr`        | References            |
| `K`         | Hover documentation   |
| `Space l a` | Code actions          |
| `Space l r` | Rename symbol         |
| `Space l f` | Format                |
| `Space l d` | Buffer diagnostics    |
| `Space l D` | Workspace diagnostics |

## Diagnostics (Trouble)

| Key         | Action                   |
| ----------- | ------------------------ |
| `Ctrl \`    | Toggle diagnostics panel |
| `Space l d` | Buffer diagnostics       |
| `Space l D` | All diagnostics          |
| `Space l s` | Symbols outline          |
| `Space l T` | LSP references           |

Inside Trouble panel:

| Key      | Action         |
| -------- | -------------- |
| `j/k`    | Navigate items |
| `Enter`  | Jump to item   |
| `Ctrl s` | Open in split  |
| `Ctrl v` | Open in vsplit |
| `Tab`    | Toggle fold    |
| `q`      | Close panel    |
